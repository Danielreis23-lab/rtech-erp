# Rtech ERP

Sistemas simples de ERP desenvolvido 
com FastAPI,  SOLAlchemy, Jinja2 e 
SQLLite para gerenciamento de pedidos,
produtos e usuários

## Funcionalidades
- Cadastro de usuário
- Login
- Cadastro de produto
- Criação de pedidos 
- Controle de status dos pedidos 
- Painel com valor pendente e faturado

## Tecnologias
- FastAPI
- SQLAlchemy
- Jnja2
- SQLLite
- Python
- HTML
- CSS
